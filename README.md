# Taller Atlas Garage - Sitio Demo

Sitio web demo ficticio para un taller mecánico. Diseño industrial/tech, responsivo, con modo claro/oscuro.

## Stack

- Next.js 14 (App Router) + TypeScript
- Tailwind CSS (modo claro/oscuro con toggle, persistido en localStorage)
- react-hook-form + zod + @hookform/resolvers
- react-icons
- next/font (Inter)

## Instalación

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Rutas

- `/` - Home
- `/servicios` - Lista detallada con acordeón
- `/nosotros` - Historia, valores, equipo, garantía de proceso
- `/promociones` - Promos con filtros
- `/casos` - Casos reales (antes/después)
- `/contacto` - Formulario + datos
- `/cotizar` - Wizard de cotización
